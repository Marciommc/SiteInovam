# Identidade Visual e UI System

## 1. Direção de Marca
*   **Personalidade:** Institucional, tecnológica, confiável, “deep tech amazônica”, pronta para escalar.
*   **Tom de Voz:** Direto, seguro, focado em valor e evidência.

## 2. Estilo UI
*   **Layout:** Grid system (12 colunas).
*   **Espaçamento:** Uso generoso de "whitespace" (respiro), especialmente no modo dark.
*   **Cards:** Bordas suaves (rounded-xl ou 12-16px).
*   **Iconografia:** Ícones lineares e modernos (Lucide React ou Heroicons).
*   **Interações:** Micro-animações leves (hover states, fade-ins, parallax sutil).

## 3. Tipografia
*   **Família Principal:** **Inter** (Google Fonts).
*   **Pesos:** Light (300), Regular (400), Medium (500), Semibold (600), Bold (700).
*   **Fallback:** system-ui, -apple-system, sans-serif.

## 4. Paleta de Cores

### Base (Dark Mode)
*   **Background:** `#0A0A0B` (Preto Profundo / Zinc 950)
*   **Surface:** `#18181B` (Zinc 900)
*   **Border:** `#27272A` (Zinc 800)

### Primary (Tech Blue)
*   **Main:** `#2563EB` (Blue 600) - Botões, Links, CTAs.
*   **Hover:** `#1D4ED8` (Blue 700)
*   **Light:** `#DBEAFE` (Blue 100) - Fundos sutis.

### Secondary (Tech Green)
*   **Main:** `#10B981` (Emerald 500) - Destaques de sucesso, inovação, sustentabilidade.
*   **Hover:** `#059669` (Emerald 600)

### Neutral (Text)
*   **Title:** `#FAFAFA` (Zinc 50)
*   **Body:** `#D4D4D8` (Zinc 300)
*   **Muted:** `#A1A1AA` (Zinc 400)

## 5. Componentes Essenciais
*   **Navbar:** Transparente com blur (glassmorphism), CTA fixo "Seja parceiro".
*   **Hero Section:** Destaque visual (vídeo ou gradiente), Headline H1 impactante.
*   **Cards de Projetos:** Exibição clara de "TRL", "Estágio" e "Impacto".
*   **Footer:** Robusto, com dados fiscais (CNPJ), endereço e links legais.
